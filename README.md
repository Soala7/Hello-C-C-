## 🪟 Windows Installation Guide

### ✅ Step 1: Install MinGW-w64
1. Go to [MinGW-w64 Downloads](https://www.mingw-w64.org/downloads/).
2. Download the latest installer.
3. Run the setup and keep the **default settings**.

---
